**题目大意** 

合并 2 个有序链表

**解题思路** 

按照题意做即可。

**代码** 

```go
package leetcode

/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
func mergeTwoLists(l1 *ListNode, l2 *ListNode) *ListNode {
 if l1 == nil {
  return l2
 }
 if l2 == nil {
  return l1
 }
 if l1.Val < l2.Val {
  l1.Next = mergeTwoLists(l1.Next, l2)
  return l1
 }
 l2.Next = mergeTwoLists(l1, l2.Next)
 return l2
}
```
